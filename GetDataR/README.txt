==================================================================
Human Activity Recognition Using Smartphones Dataset
Version 1.0
==================================================================

The dataset includes the following files:
=========================================

- 'README.txt'

-  data_group.txt : tidy data set with the average of each variable for each activity and each subject

- 'run_analysis.R': R script allows create two new data sets: data_group and data_mean_std

- 'features_info.txt': Original feature information about whole set of the variables

- 'features_group.txt': List of all features for the data set data_group.txt

- 'features_mean_std.txt': List of all features for the data set data_mean_std which can be created by script


References:
===========

[1] Davide Anguita, Alessandro Ghio, Luca Oneto, Xavier Parra and Jorge L. Reyes-Ortiz. Human Activity Recognition on Smartphones using a Multiclass Hardware-Friendly Support Vector Machine. International Workshop of Ambient Assisted Living (IWAAL 2012). Vitoria-Gasteiz, Spain. Dec 2012
